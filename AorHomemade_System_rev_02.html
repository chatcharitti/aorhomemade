ข้อผิดพลาดด้านสภาพแวดล้อม

คุณกำลังเปิดไฟล์นี้ผ่านโปรโตคอล file:/// ซึ่งละเมิดนโยบายความปลอดภัย CORS ของเบราว์เซอร์ ทำให้ระบบไม่สามารถโหลดโมดูลและฐานข้อมูลได้

วิธีแก้ไข:
กรุณารันไฟล์นี้ผ่าน Local Web Server เช่น:
- VS Code: ติดตั้งและใช้ "Live Server"
- Terminal: รัน python -m http.server 8000
- Node.js: รัน npx serve
